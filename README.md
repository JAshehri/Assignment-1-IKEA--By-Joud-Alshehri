# Dataset Description

## The dataset was obtained from Kaggle:
https://www.kaggle.com/datasets/ranjithpanicker/ikea-global-product-catalog-and-price-2026

## Dataset Size

Number of rows: 401,046

Number of columns: 18

## Features Description

The dataset contains detailed information about IKEA products across multiple countries. Key features include:

unique_id / product_id: Unique identifiers for each product

product_name / product_type: Name and type of the product

product_measurements: Product dimensions and size

product_description: Textual description of the product

main_category / sub_category: Product classification

product_rating / product_rating_count: Customer ratings and number of reviews

price / currency: Product price and its currency

online_sellable: Whether the product is available online

country: Country where the product is listed

## Purpose of the Dataset

The dataset is used to perform Exploratory Data Analysis (EDA) in order to:

Understand market differences across countries

Analyze pricing strategies and purchasing power

Explore customer behavior and satisfaction

Identify popular product categories globally

## Key Insights Summary

Significant variation exists in prices across countries

Storage and organization products are the most popular globally

Customer satisfaction is generally high

Price does not strongly correlate with customer ratings

## Notes

Prices were converted into Saudi Riyal (SAR) for consistency

Missing and inconsistent values were cleaned before analysis
